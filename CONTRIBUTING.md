# Contributing

Thanks for considering a contribution.

## Setup

```bash
npm install
npm run dev
```

## Development Rules

- Keep the public API small and typed.
- Preserve attribution to Liquid Glass Studio and Charles Yin.
- Avoid visual regressions in the demo.
- Keep animations smooth at desktop and mobile sizes.
- Run `npm run build` before opening a pull request.

## Pull Requests

Please include:

- A short description of the change
- Before and after screenshots or screen recordings for visual changes
- Any browser notes if WebGL behavior changed

## Commit Style

Use concise commit messages in this format:

```text
feat: add size prop
fix: prevent pointer capture leak
docs: improve Next.js example
```
