# Code of Conduct

This project is for practical, respectful collaboration.

## Expected Behavior

- Be direct, kind, and constructive.
- Assume good intent.
- Keep feedback focused on the work.
- Respect different experience levels and backgrounds.

## Unacceptable Behavior

- Harassment, threats, or personal attacks
- Discriminatory language
- Publishing private information
- Repeated off-topic disruption

## Reporting

Open a private security or maintainer contact channel if one is available. If this repository does not list one yet, open an issue with only the non-sensitive context and ask for a private contact method.
